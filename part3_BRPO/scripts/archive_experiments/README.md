# archive_experiments

This directory stores historical one-off experiment launchers and legacy entry scripts.
Current live script entrypoints stay at `scripts/` top level; archived runners are grouped by topic here to reduce noise in the live workspace.

Subdirs:
- `a1/`: historical A1 / target-side compare runners
- `g/`: historical G-BRPO compare / smoke runners
- `topology/`: historical topology compare runners
- `legacy_entry/`: superseded top-level training entrypoints
- `stageA/`: historical StageA/abs-prior compare launchers
