# Part 2 Workspace

This directory is reserved for Part 2 (Unposed Sparse Reconstruction) experiments using RegGS.

## Layout

- `reggs/`: repository checkout will go here later
- `data/re10k_1_sparse/`: sparse, unposed input prepared from Re10k-1
- `outputs/reggs_init/`: RegGS intermediate and initialization outputs
- `outputs/gs/`: downstream 3DGS outputs
- `scripts/`: helper scripts for data prep / conversion / evaluation
